有效的对话技巧

示例 1: 清晰的问题

用户：请帮我写一个Python函数，用于计算两个数字的最大公约数
Claude：我来为你写一个计算最大公约数的Python函数： 
def gcd(a, b):
    """
    计算两个数字的最大公约数
    使用欧几里得算法
    """
    while b:
        a, b = b, a % b
    return a

# 使用示例
print(gcd(48, 18))  # 输出: 6

示例 2: 代码解释

用户：请解释这段代码的工作原理
Claude：这个函数使用了欧几里得算法：
  • 当b不为0时，继续循环
  • 每次迭代中，将a设为b，b设为a除以b的余数
  • 当b为0时，a就是最大公约数

常见错误

⚠️ 避免这些常见错误：

  • 问题过于宽泛：如"帮我写个程序"而不说明具体功能
  • 缺少上下文：不提供相关的背景信息或约束条件
  • 盲目信任：不验证生成的代码就直接使用
  • 忽略错误处理：不考虑边界情况和异常处理
