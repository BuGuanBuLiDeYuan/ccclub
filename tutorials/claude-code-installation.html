什么是Claude Code

Claude Code 是 Anthropic 开发的智能编程助手，直接运行在您的终端中。它能够：

  • 理解代码库：分析整个项目结构和代码逻辑
  • 自然语言编程：通过对话方式编写和修改代码
  • 执行任务：运行测试、处理Git工作流、编辑文件
  • 智能建议：提供代码优化和最佳实践建议

💡 核心优势

Claude Code 使用最新的 Sonnet 4 和 Opus 4.1 模型，支持128K上下文，能处理大型代码库和复杂项目。

系统要求

🖥️ 操作系统

  • macOS 10.15+ (推荐)
  • Linux Ubuntu 20.04+/Debian 10+
  • Windows 10+ (需要 WSL 或 Git Bash)

💾 硬件要求

  • 最少 4GB RAM
  • 稳定的网络连接
  • 终端访问权限

📄 账户要求

  • Claude Pro 或 Team 订阅
  • 或有效的 Anthropic API 账户
  • 活跃的计费账户

安装方法

方法一：NPM 安装（推荐）

1. 配置 npm 全局安装目录

# 创建全局npm目录
mkdir -p ~/.npm-global

# 配置npm前缀
npm config set prefix ~/.npm-global

# 添加到PATH环境变量
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

2. 安装 Claude Code

# 全局安装Claude Code CLI
npm install -g @anthropic/claude-code

⚠️ 重要提示

避免使用 sudo npm install -g，这会导致权限问题和安全风险。

方法二：原生二进制安装

macOS/Linux 安装脚本

# 安装稳定版本
curl -fsSL https://claude.ai/install.sh | bash

# 安装最新版本
curl -fsSL https://claude.ai/install.sh | bash -s latest

Windows PowerShell 安装

# 安装稳定版本
irm https://claude.ai/install.ps1 | iex

# 安装最新版本
& ([scriptblock]::Create((irm https://claude.ai/install.ps1))) latest

# 检查安装状态

常见问题

Q: 安装后无法找到 claude 命令

A: 检查PATH环境变量配置，确保npm全局bin目录已添加到PATH中。

# 检查npm全局bin路径
npm config get prefix

# 手动添加到PATH
export PATH="$(npm config get prefix)/bin:$PATH"

Q: 权限认证失败

A: 确保您的Anthropic账户有有效的Pro或Team订阅，并且计费状态正常。

Q: 如何跳过权限询问

A: 使用以下命令快速启动：

claude --dangerously-skip-permissions
